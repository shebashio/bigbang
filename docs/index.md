# Big Bang

Big Bang is a declarative, GitOps-driven delivery tool for deploying and managing Department of Defense (DoD) hardened and approved packages on Kubernetes. It packages a DevSecOps platform as an umbrella Helm chart and uses Flux to reconcile declared state into a cluster.
